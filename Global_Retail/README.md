# Global Retail Performance and Operations Intelligence Dashboard

## Project Overview

This project is a comprehensive Power BI report developed to provide a unified, executive level view of global retail performance across sales, customers, regions, and operations.

Global Retail Corporation operates across multiple international markets with both physical and online channels. Data exists across separate systems, making it difficult for leadership to gain a complete view of performance.

This report consolidates that data into a single, interactive experience that enables fast, informed decision making.

The solution is structured into three focused report pages:

- Executive Overview  
- Customer and Market Analysis  
- Regional and Operations Performance  

Each page is designed to answer specific business questions while maintaining a consistent and intuitive user experience.

---

## Business Problem

Executive leadership lacked a centralized reporting solution that integrates:

- Sales and profitability  
- Customer behavior  
- Returns and operational performance  
- Regional and market level insights  

As a result, decisions around growth, cost control, and operational improvements were made without full visibility into key performance drivers and risks.

---

## Project Objectives

- Build an interactive Power BI report integrating multiple data sources  
- Design a scalable and optimized data model  
- Provide clear visibility into sales, profit, customers, and operations  
- Identify underperforming regions, products, and customer segments  
- Enable data driven decision making through dynamic reporting  

---

## Key Business Questions

- How are sales, profit, and profit margin trending over time?  
- Which regions and customer segments generate the highest profitability?  
- How do return rates impact overall performance?  
- Are shipping methods affecting delivery efficiency?  
- Which managers or regions are outperforming or underperforming?

## Dataset

**Source:** Global Superstore Dataset  

### Tables Used:
- Orders 
- Returns  
- People (Regional Managers)  
- Date Table (created for time intelligence)  

### Key Fields:
- Order ID  
- Customer ID  
- Order Date  
- Sales and Profit  
- Category and Sub Category  
- Market and Region  
- Shipping Mode  
- Return Status  

---

## Data Model

- Star schema design implemented  
- Fact table: Orders  
- Dimension tables: Customers, Products, Region, Date, Returns  
- Relationships established using primary and foreign keys  
- Dedicated Date table created to support time intelligence  

## Data Preparation

- Cleaned and transformed data using Power Query  
- Removed duplicates and inconsistencies  
- Standardized categorical fields  
- Validated key metrics (Sales, Profit)  
- Created calculated columns and measures  
- Built structured relationships across tables  

## Technical Implementation

### Tools & Technologies
- Power BI  
- Power Query  
- DAX  

### Features Implemented
- Interactive slicers (Year, Market, Region, Segment, Category)  
- KPI cards and dynamic measures  
- Time intelligence calculations  
- Drill down and cross filtering  
- Multi page report navigation  
- Optimized visuals for executive consumption  

## Key Insights

- Sales and profit show consistent growth with seasonal patterns  
- Technology category is the most profitable, while Furniture underperforms  
- Consumer segment drives the highest share of revenue  
- A small group of customers contributes disproportionately to total sales  
- USCA and Asia Pacific are the top performing markets  
- Standard shipping generates the most sales but has longer delivery times  
- Return rates vary significantly across categories, indicating operational gaps  
- Delivery performance is relatively consistent across regions
